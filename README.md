## 🚀 Tech

- ⚛️ React-Native
- 🖇 Expo
- ❕ Yarn

## ✋🏻 Prerequisites

- [Node.js](https://nodejs.org/)
- [Expo-cli](https://expo.io/tools#cli)
- [Yarn](https://yarnpkg.com/)

## 🔥 Install & Execute

1. Clone the repo;
2. Open the cloned repo;
3. Run `yarn` to install the dependecies;
4. Run `yarn start` or `expo start` to start the app;
